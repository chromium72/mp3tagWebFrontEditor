# MP3 Tag Editor

ブラウザ上で完結する、軽量で高速なMP3タグ（ID3v2）エディタです。
サーバーへのファイルアップロードは一切不要。PC内のローカルフォルダを直接読み込み、編集したタグを元のファイルに直接上書き保存します。

## ✨ 特徴 (Features)

- **完全ローカル処理**: File System Access API を使用。ファイルはサーバーに送信されず、安全にローカルで処理されます。
- **一括編集**: 複数のファイルを同時に選択し、アーティスト名やアルバム名などを一度に書き換えることができます。
- **カバーアートの最適化**: 巨大な画像をアップロードしても、ブラウザ側で自動的にリサイズ・圧縮（最大800px / JPEG）を行い、MP3ファイルの肥大化を防ぎます。
- **文字化け自動修復**: 日本語環境の古いMP3ファイルで発生しがちな「Shift-JIS由来の文字化け」を検知し、自動的に復元して表示します。
- **モダンなUI**: Tailwind CSSによる洗練されたデザイン。ウィンドウサイズに応じたレスポンシブなレイアウトに対応しています。

## ⚠️ 動作環境 (Compatibility)

このアプリケーションは、ローカルファイルへの直接アクセスを可能にする「File System Access API」を使用しています。そのため、以下のブラウザでのみ動作します。

- **必須要件**: PC版 Google Chrome または Microsoft Edge
- **非対応**: Safari, Firefox, およびすべてのスマートフォン・タブレットのブラウザ（API未対応のためブロック画面が表示されます）。

## 🛠 技術スタック (Technologies)

ビルドツールやNode.js環境は不要です。単一のHTMLファイルとして動作します。

- **UI**: HTML5, Vanilla JavaScript, Tailwind CSS (CDN)
- **ID3読み込み**: [jsmediatags](https://github.com/aadsm/jsmediatags) (CDN)
- **ID3書き込み**: [browser-id3-writer](https://github.com/egoroof/browser-id3-writer) (v4.4.0 / CDN)

## 🚀 使い方 (How to Use)

1. このリポジトリをダウンロードするか、公開されているURL（GitHub Pagesなど）に **Google Chrome** でアクセスします。
2. 画面右上の 「**フォルダを開く**」 をクリックし、MP3ファイルが含まれているPC内のフォルダを選択します。
3. ブラウザから「ファイルの変更の保存」に対する許可を求められるので、「**変更の保存を許可する**」 を選択します。
4. 左側のリストから編集したいファイルを選択します（`Ctrl`キーを押しながらクリック、またはチェックボックスで複数選択が可能）。
5. 中央のフォームでタイトル、アーティスト、カバーアートなどを編集します。
6. 右上の 「**保存**」 ボタンをクリックすると、元のMP3ファイルに直接上書き保存されます。

## 📝 制限事項 (Limitations)

- フォルダの読み込みは、選択した階層のみが対象です。サブフォルダ内のMP3ファイルは再帰的に読み込まれません。
- ID3タグの書き込みは ID3v2 形式で行われます。

## 🤝 開発・貢献について

このプロジェクトは単一の `index.html` にすべてのロジックが含まれています。
ローカルで改修を行う際は、ファイルを編集してブラウザでリロードするだけで変更を確認できます。

リポジトリをクローン: `git clone <repository-url>`

`index.html` を Chrome で開く。
